# File Tree Smart

[![VS Code](https://img.shields.io/badge/VS%20Code-1.90+-blue.svg)](https://code.visualstudio.com/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.5.3-blue.svg)](https://www.typescriptlang.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](https://github.com/PlayRood32/file-tree-smart/LICENSE)

Professional VS Code extension for generating clean Markdown file trees.

Supports smart icon rendering, `.gitignore` parsing (root + nested), and deterministic output.
Created by **PlayRood**. GitHub: **PlayRood32**.

## âœ¨ Features

- Generate Markdown tree with icons
- Generate Markdown tree without icons
- Quick Pick mode selector
- Explorer folder right-click integration
- Command Palette integration
- Recursive traversal with deterministic sort (directories first)
- `.gitignore` support from root and nested folders
- Built-in fallback ignore list when `.gitignore` is missing
- Large icon map for filenames, patterns, and extensions
- Empty folder handling after filtering
- Clean final output template suitable for docs and PRs

## ğŸš€ Installation

### From VSIX

```bash
# 1) Install dependencies
npm install

# 2) Compile
npm run compile

# 3) Package extension
npx vsce package
```

Then install the generated `.vsix` file in VS Code:

1. Open Command Palette (`Ctrl+Shift+P`)
2. Run `Extensions: Install from VSIX...`
3. Select the generated `.vsix` file

### Prerequisites

- VS Code `^1.90.0`
- Node.js 18+
- npm 9+

## ğŸ“– Usage

### Folder Context Menu

1. Right-click any folder in Explorer
2. Choose one of:
   - `Generate Markdown Tree (with icons)`
   - `Generate Markdown Tree (no icons)`
   - `Generate Markdown Tree...`

### Command Palette

1. Open Command Palette
2. Run one of the same commands
3. If no folder context is provided, choose a workspace folder

The output opens in a new untitled Markdown editor.

## ğŸ§ª Troubleshooting

<details>
<summary><b>Commands are not visible in right-click menu</b></summary>

- Make sure you are right-clicking a folder (not a file)
- Ensure the extension is installed and enabled
- Reload VS Code window after install/update
</details>

<details>
<summary><b>Generated tree is empty</b></summary>

- Check `showHiddenFiles` setting
- Check `.gitignore` patterns
- Try setting `respectGitignore` to `false` for verification
</details>

<details>
<summary><b>VSIX packaging fails</b></summary>

- Ensure `npm run compile` passes
- Ensure `vsce` is installed (`npm i -D @vscode/vsce`)
- Retry: `npx vsce package`
</details>
